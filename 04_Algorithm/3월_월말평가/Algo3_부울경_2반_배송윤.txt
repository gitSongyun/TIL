빅오표기법이란 시간복잡도를 나타내는 대표적인 방법이며, 알고리즘을 수행하는데 걸리는 시간을 설명하는 계산 복잡도이다. 
시간복잡도의 점근적 상한을 나타내기 때문에 최악의 경우 이 정도 시간이 소요 된다는 것을 의미한다. 
데이터의 입력값이 충분히 크다고 가정하기 때문에 상대적으로 영향력이 작은 차수의 항은 무시하고, 최고차항을 계수없이 표기한다. 
따라서 f(n) = 7n+4의 경우 빅오 표기법으로 O(n) 으로 나타낼 수 있다. 

2. 𝑇 (𝑛) = 𝑇 (𝑛−1) + 1, T(0) = 1

sol>
T(n) 
= T(n-1) + 1
= T(n-2) + 2
= T(n-3) + 3
= T(n-k) + k
= T(0) + n         (n=k 일 때)
= 1 + n

∴ 빅오 표기법으로 표현하면 O(n)
