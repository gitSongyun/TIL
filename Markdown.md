### 수평선(Horizontal line)

`---` 하거나 `<hr>` 하면 밑줄 생김

### 코드블럭(code block)

1. 코드박스 (``` + Ent)

   ``` python
   class = '부울경 2반'
   ```

   

2. 인라인코드 (백킷 사이에 코드 작성)
   - `print('부울경 2반')`

  - [삼성](https://samsung.com)

### 텍스트 강조(Text Emphasis)

- 굵게 : **부울경**, __부울경__
- 기울임 : *부울경*
- 취소선 : ~~부울경~~

*대신 _(underbar)로 대체 할 수 있다. 

---

### 인용문(Blockquotes)

- `>` , `>>` 사용

>대충 흑백사진에 글쓰면 명언같다.
