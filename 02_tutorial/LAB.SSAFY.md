`--global` : 전역, 현재위치 뿐만 아니라 전체가 참조할 수 있는

`--local` : 지역, 현재 위치에서만 참조하는

2. `git init`
3. `git config --local user.name '<username>'` (한글이름)

5. `git remote add origin <git lab url>`
